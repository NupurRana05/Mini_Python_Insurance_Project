Mini Python Insurance Data Analysis Project

1. Introduction

This project is a Mini Insurance Data Analysis project built using Python.

The objective of this project is to perform data cleaning, exploratory data analysis (EDA), visualization, correlation analysis, and generate a Probability Distribution Function (PDF) report with charts.

The dataset contains customer insurance information, including age, BMI, smoking status, region, and insurance charges.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

2. Project Objectives

* Understand the structure and quality of insurance data

* Perform data cleaning and preprocessing

* Conduct exploratory data analysis (EDA)

* Analyze relationships between features and insurance charges

* Visualize key business insights using charts

* Generate Probability Distribution Function (PDF) plots

* Export analysis results into a professional PDF report
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
3. Technologies & Tools Used

* Python

* Pandas – Data manipulation

* NumPy – Numerical operations

* Matplotlib – Data visualization

* Seaborn – Statistical plots

* ReportLab – PDF report generation

* Jupyter Notebook
